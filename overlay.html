Ubuntu Desktop 22.04 LTS Tutorial

WiseTechLTS

Follow this step-by-step guide based on official Ubuntu documentation to install, configure, secure, and connect to your Ubuntu Desktop 22.04 system.

In this tutorial, we will walk you through the complete process of installing Ubuntu Desktop 22.04 LTS.

You will also learn about enabling Ubuntu Pro, activating the Ubuntu Security Guarantee (USG) features, performing a DISA STIG compliance test, and setting up secure SSH key-based access.

Pre-installation Preparation:

Ensure your system meets the following minimum requirements (as per Ubuntu 22.04 LTS documentation):

  • 2 GHz dual‑core processor
  • 4 GB RAM (8 GB recommended for optimal performance)
  • 25 GB free disk space (SSD recommended)
  • Bootable USB drive or DVD

Download the Ubuntu Desktop 22.04 ISO from the official Ubuntu website. Verify the ISO's checksum as instructed in the release notes to ensure file integrity. Then, create a bootable USB drive using Rufus (Windows) or Etcher (macOS/Linux).

Installing Ubuntu Desktop 22.04:

1. Insert your bootable USB drive and restart your computer.
2. Access your BIOS/UEFI settings by pressing F2, F12, DEL, or Esc during startup.
3. Change the boot order to prioritize your USB drive, save the settings, and reboot.

When the Ubuntu installer loads, choose “Try Ubuntu” to test the live environment or “Install Ubuntu” to begin installation.

  • Select your preferred language and keyboard layout.
  • Choose your time zone and regional settings.
  • Select “Normal Installation” for a complete setup including documentation and media support.
  • For a clean install, choose “Erase disk and install Ubuntu.” Otherwise, select manual partitioning for dual-boot or custom setups.
  • Create your user account, set your password, and complete the installation process.

For further details, refer to the Ubuntu 22.04 Installation Guide.

Post-installation Initial Setup:

After installation, restart your system and log in with your new account. Open a terminal window (Ctrl+Alt+T) and update your system by running: 

sudo apt update && sudo apt upgrade -y

This ensures your system is updated with the latest security patches and software improvements.

Enabling Ubuntu Pro:

Ubuntu Pro extends your system's security with enhanced patches and compliance tools.

  1. Visit ubuntu.com/pro and register for a free token.
  2. In the terminal, run: 

    sudo pro attach <your-token>
  3. Verify activation with: 

    sudo pro status

This process integrates extended security updates into your system.

Enabling Ubuntu Security Guarantee (USG) Features:

With Ubuntu Pro active, your system can utilize USG features for an extra layer of security. Check these services by running: 

sudo pro status --services

Install any additional recommended security packages to meet enterprise-grade protection standards.

Testing the System:

Verify the integrity and performance of your system: 

sudo systemctl status

Additionally, check log files in /var/log for errors and consult the Ubuntu Security Guide for further hardening recommendations.

Closing:

That concludes our comprehensive tutorial on installing Ubuntu Desktop 22.04 LTS and configuring it with Ubuntu Pro and USG features. Your system is now updated, secure, and optimized for performance.

DISA STIG Compliance Test:

To adhere to rigorous security standards, perform a DISA STIG compliance test. The DISA Security Technical Implementation Guide (STIG) provides detailed instructions for system hardening.

Follow these steps:

  1. Install Required Tools:
    sudo apt install openscap-utils scap-workbench -y
  2. Download the Ubuntu 22.04 STIG Profile:
    Obtain the latest STIG profile (e.g., ssg-ubuntu2204-ds.xml) from the official DISA website or SCAP Security Guide repository.
  3. Run the Compliance Scan:
    
sudo oscap oval eval --results results.xml --report report.html /usr/share/openscap/ssg/content/ssg-ubuntu2204-ds.xml
  4. Review the Report:
    Open report.html in a browser to review the findings and recommended remediation steps.

SSH Connection Setup Using SSH Keys:

To securely connect between two Ubuntu desktop computers using SSH keys, follow these steps:

Step 1: Install OpenSSH (if not already installed)
On both machines, run: 

sudo apt update && sudo apt install -y openssh-server

Check the status of the SSH service: 

sudo systemctl status ssh

If the service is inactive, enable and start it: 

sudo systemctl enable --now ssh

Step 2: Generate SSH Key on the Client Machine
On the client, run: 

ssh-keygen -t rsa -b 4096

Accept the default location (~/.ssh/id_rsa) and optionally set a passphrase.

Step 3: Copy the SSH Public Key to the Server
Run: 

ssh-copy-id user@10.0.0.227

(Replace user with your server username.) If ssh-copy-id is unavailable, run: 


cat ~/.ssh/id_rsa.pub | ssh user@10.0.0.227 "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys && chmod 600 ~/.ssh/authorized_keys"

Step 4: Test SSH Login Without Password
From the client, run: 

ssh user@10.0.0.227

You should log in without being prompted for a password.

Step 5: (Optional) Disable Password Authentication
On the server, edit /etc/ssh/sshd_config to set: 


PasswordAuthentication no
PubkeyAuthentication yes

Then restart SSH: 

sudo systemctl restart ssh

Step 6: Verify and Troubleshoot
If issues occur, check: 

sudo systemctl status ssh

Ensure that ~/.ssh has permissions set to 700 and authorized_keys is 600: 

ls -lah ~/.ssh

Allow SSH through the firewall if necessary: 


sudo ufw allow OpenSSH
sudo ufw enable

This completes the SSH key setup process.
