Ubuntu Desktop 22.04 Installation & Configuration Tutorial

WiseTechLTS

Introduction:
Welcome to WiseTechLTS. In today’s tutorial, we’ll guide you through installing Ubuntu Desktop 22.04 and walk you through the post‑install configuration steps—including enabling Ubuntu Pro, activating the Ubuntu Security Guarantee (USG) features, and testing your system for optimal performance.

Pre-installation Preparation:
Before we begin, ensure your hardware meets the minimum requirements:
• 2 GHz dual‑core processor
• 4 GB RAM
• 25 GB free disk space
• Bootable USB or DVD
Download the Ubuntu Desktop 22.04 ISO from the official website and create a bootable USB drive using a trusted tool such as Rufus or Etcher.

Installing Ubuntu Desktop 22.04:
Insert your bootable USB drive and restart your computer. Access your BIOS/UEFI settings (typically by pressing F2, F12, or DEL) and set the USB drive as the primary boot device. Once the Ubuntu welcome screen appears, select “Try Ubuntu” or “Install Ubuntu.” Choose your language, keyboard layout, time zone, and opt for “Normal Installation.” For a clean setup, select “Erase disk and install Ubuntu” (or use manual partitioning if you prefer advanced configurations). Follow the prompts to create your user account and complete the installation.

Post-installation Initial Setup:
After installation, restart your system and log in. Open a terminal (Ctrl+Alt+T) and update your package repositories by running:
sudo apt update && sudo apt upgrade -y
This ensures your system has the latest updates and security patches.

Enabling Ubuntu Pro:
Enhance your system’s security by enabling Ubuntu Pro. Visit ubuntu.com/pro to register and obtain a free token. In the terminal, execute:
sudo pro attach <your-token>
Then verify the activation by running:
sudo pro status

Enabling USG Features:
With Ubuntu Pro active, confirm that the Ubuntu Security Guarantee (USG) features are enabled. Run:
sudo pro status --services
If prompted, install any additional security packages to ensure your system meets enterprise-level security standards.

Testing the System:
Test your system’s stability and security by validating the integrity of installed packages and checking security logs. Run:
sudo systemctl status
to verify that all essential services are running correctly. For further hardening, refer to the Ubuntu Security Guide.
